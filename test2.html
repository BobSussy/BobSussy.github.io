About

Delorem Ipsum Online Store 是作为 IT1566 - App Development Project module 的小组项目的一部分创建的。

Delorem Ipsum 是一个基于 Web 的 Python 应用程序，用作在线药店。

我主要负责这个项目的管理界面。我完成的功能如下：

  • 管理仪表板
    • 库存不足的商品
    • 待交易
    • 最畅销/浏览次数最多的商品
  • 产品目录
    • 添加新产品
    • 编辑现有产品
    • 添加库存
    • 导出产品信息（excel表格）
  • 产品统计
    • 突出显示低于/接近库存价值阈值的产品
    • 产品统计的图形化视图
    • 出口产品统计（Excel表格）
  • 交易
    • 查看交易信息
    • 将交易标记为完成/不完整
    • 导出交易信息（excel表格）
  • 搜索栏 – 过滤产品
